💻 PROYEK

Proyek-proyek yang dikerjakan selama studi Sistem Informasi di UNIKMA:

📦 Sistem Manajemen Inventaris Toko

Aplikasi web untuk mengelola stok barang, transaksi penjualan, dan laporan keuangan menggunakan PHP, MySQL, dan Bootstrap. Dilengkapi notifikasi stok menipis otomatis.

🏥 Sistem Informasi Klinik Sederhana

Aplikasi manajemen pasien, rekam medis, dan jadwal dokter untuk klinik kecil. Dibangun menggunakan CodeIgniter 4 dan MySQL untuk memudahkan administrasi digital.

📊 Dashboard Visualisasi Data Akademik

Menampilkan data nilai dan kehadiran mahasiswa dalam grafik interaktif menggunakan Chart.js dan PHP. Membantu dosen memantau performa mahasiswa secara real-time.

🍳 Aplikasi Resep Masakan Digital

Aplikasi web mobile-first untuk menyimpan, mencari, dan berbagi resep masakan menggunakan HTML, CSS, JavaScript, dan Firebase. Terinspirasi dari hobi memasak!

🎨 Desain UI/UX Aplikasi E-Learning

Prototype antarmuka aplikasi pembelajaran daring untuk siswa MTs/MA menggunakan Figma, menerapkan prinsip UX Research dan high-fidelity prototype.
